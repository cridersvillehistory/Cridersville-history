# Cridersville Historical Society Website (Static)

This is a simple static website intended to be hosted for free using GitHub Pages.

## Replace the banner image
- `assets/hero.svg` is a placeholder.
- Replace it with a real photo if you want. You can keep the filename `hero.svg` or update `assets/styles.css` to point to `hero.jpg`.

## Contact form
This site uses a `mailto:` link by default (opens the visitor's email app). If you want a true web form, connect to Formspree or Netlify Forms.

## Publishing
Follow the step-by-step instructions in the chat for GitHub Pages.
